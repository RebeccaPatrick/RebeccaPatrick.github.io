Game Controllers

Wearable Nintendo switch controllers pink and green Splatoon version. For this costume I challenged myself to make a Halloween costume within a budget. Inspired by a wall feature in a social space I decided to make a set of game controller costumes. I primarily used cardboard that was destined for the bin.

The buttons have wire spun into springs, so they can be pressed.

The joystick is on a simple hair tie mechanism that I created, making it possible to move around and then springing back to the centre just how Nintendo game controller would react.
